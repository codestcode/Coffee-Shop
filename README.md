# ☕ BrewCraft Coffee – E-commarce Coffee Website

A modern, premium coffee shop frontend built to deliver a smooth, immersive, and visually engaging user experience.

This project focuses on transforming a coffee brand into a polished digital storefront , combining elegant UI, smooth animations, and interactive elements like 3D rendering.

---

## => Overview

BrewCraft is a frontend coffee shop template designed to simulate a real e-commerce experience while remaining flexible and client-ready.

It can be easily customized for different brands and business needs, making it suitable for real-world client projects.

It includes:

- Landing page with strong brand storytelling
- Product catalog with filters & sorting
- Product detail pages
- Shopping cart system
- Mock checkout flow

---

## ✨ Features

- Clean, modern UI with premium feel
- Smooth animations using Framer Motion
- State management with React Context
- Persistent shopping cart (localStorage)
- Product filtering & sorting
- Multi-page routing experience
- Interactive 3D coffee bean (React Three Fiber)
- Built as a reusable and customizable template for clients

---

## Customization & Client Adaptation

This project is built with flexibility in mind and can be adapted for real client use:

- Full branding customization (colors, typography, assets)
- Replace products with real client inventory
- Modify pages and sections based on business needs
- Payment gateway integration (Stripe, PayPal, etc.) can be added if required
- Backend/API integration for real data and orders
- Custom feature additions based on client requirements

---

## 🛠 Tech Stack

- React 19 + TypeScript
- Vite
- Tailwind CSS
- Framer Motion
- React Router DOM
- @react-three/fiber
